Data Sorting and Merging

Once you have clean data, you often need to organize it to find the most important information. Sorting allows you to see the highest and lowest values in a category. Creating new metrics by combining existing ones helps you see "the big picture" of a player's contribution.

Key Concepts:

  • Sorting Data: Using df.sort_values(by='ColumnName') to organize data.
  • Ascending vs. Descending: Use ascending=False to see the highest values at the top.
  • Combining Columns: Creating new statistics like "STOCKS" (Steals + Blocks) to measure defensive impact.
  • Previewing Data: Using .head() to focus on the top performers.

In this activity, you'll use Python and the pandas library to identify the top scorers and defensive specialists in the NBA using data from the 2024-2025 season.
