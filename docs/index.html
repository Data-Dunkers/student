Data Dunkers

Welcome to the Data Dunkers course for students. In this course, you'll learn how to analyze basketball data to find interesting insights about players and teams.

How to use this site

This site contains a growing set of Data Dunkers activities. To help you choose where to start, activities may include short labels:

  • Intro – A good starting point. Low setup, focused on understanding or interpretation.
  • Core – Part of the main learning path. Builds skills used in later activities.
  • Stretch – Optional or exploratory. Best for extension or curiosity.

You can follow the suggested progression or jump to any activity based on your goals.

Tools

These activities are web pages or Jupyter Notebooks. Both types are interactive, but the notebook versions allow for more flexibility and exploration.

Jupyter Notebooks are interactive workspaces that can include text, code, and visuals. These notebooks make it easier to explore data step by step, test ideas, and build visualizations. The notebooks work in a web browser, but they can also be run on your own computer by installing VS Code or a similar development environment.

Using Jupyter Notebooks

Each activity opens in a notebook that is already set up with instructions, starter code, and space to experiment. Go through the notebook from top to bottom, running cells as you go. You can adjust variables, change chart types, or explore additional data. Mistakes are normal, you can re-run cells or reset sections as needed.

Saving Work, Adding Notes, and Sharing With Teachers

Both web pages and notebooks include places where you can answer questions and add your thoughts. In notebooks you can also insert new text cells anywhere to capture ideas as you work.
