Data Dunkers - Lesson Points Analysis

This page is intended for internal teacher use only. It provides a complexity score (Points) for each lesson to help with grading, student pacing, or curriculum planning.

Scoring Factors (1-10 Scale)

  • Technical Depth: Amount of Python/Pandas code required (0 = No code, 10 = Advanced logic/looping).
  • Cognitive Load: Complexity of the abstract concepts (e.g., probability vs. simple reading).
  • Dataset Complexity: Size and "messiness" of the data (e.g., raw CSV vs. pre-cleaned metrics).
  • Output Complexity: Difficulty of the final visualization or decision-making requirement.
